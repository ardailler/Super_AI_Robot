## Entrée du réseau 

Tableau à deux valeurs: 
- Distance du mur en ligne (0 si le mur est devant le robot)
- Point cardinal de l'agent : 
    - 0 : EST
    - 1 : SUD
    - 2 : OUEST
    - 3 : NORD
    
    
 ## Action possible
 
- Rotation droite à 90°
- Rotation gauche à 90°
- Avancer


## Motivation 

Explorer le maximum l'environnement
